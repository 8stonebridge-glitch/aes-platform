# Admin Console

Back-office admin panel template for managing users, resources, configurations, and audit logs.

## App Class

`internal_ops_tool`

## Target Users

Internal teams (admins, super admins, support staff) who need full CRUD access with role-gated views across organizational resources.

## Key Capabilities

- Sidebar navigation with grouped icon sections
- Data tables with server-side filtering, sorting, and pagination
- Detail pages with tabbed content areas
- Settings and configuration management
- Audit log viewer with filterable event history
- Role-gated route protection via Clerk

## Stack

Next.js + Convex + Clerk + Vercel + XState
