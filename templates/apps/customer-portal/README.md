# Customer Portal

Self-service portal template for external customers.

## App Class

`customer_portal`

## Target Users

End customers who need to manage their account, view invoices, submit support requests, and access documentation.

## Key Capabilities

- Account summary dashboard
- Subscription and plan management
- Support ticket submission and status tracking
- Document and invoice download center
- Profile and notification preferences
- Onboarding wizard for new customers
- Org switching for multi-org accounts

## Stack

Next.js + Convex + Clerk + Vercel + XState
