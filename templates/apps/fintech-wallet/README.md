# Fintech Wallet

Financial services app template for account management, transfers, and transaction tracking.

## App Class

`fintech_wallet`

## Target Users

Fintech products serving end users who need to manage balances, send money, view transaction history, and manage beneficiaries.

## Key Capabilities

- Account overview dashboard with balance display
- Transaction list with date, type, and status filters
- Transfer form with XState-driven state machine
- Beneficiary management (add, edit, remove)
- Payment integration shell (Stripe/Paystack)
- Audit trail for all financial mutations
- Offline balance viewing via PWA
- Multi-currency support
- PIN/biometric confirmation for sensitive actions

## Stack

Next.js + Convex + Clerk + Vercel + XState
