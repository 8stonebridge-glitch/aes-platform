# Internal Ops PWA

Progressive web app template for field teams and internal operations staff.

## App Class

`internal_ops_pwa`

## Target Users

Field workers, warehouse staff, delivery teams, and operations managers who need mobile-first access with offline support.

## Key Capabilities

- PWA with offline-first data sync
- Task lists with status-based filtering
- Checklist completion flows
- Push notifications
- Photo capture and file attachment
- Location tagging
- Bottom tab navigation optimized for mobile

## Stack

Next.js + Convex + Clerk + Vercel + XState
