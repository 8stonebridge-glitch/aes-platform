# Marketplace Lite

Two-sided marketplace template for listing, searching, and transacting items or services.

## App Class

`marketplace`

## Target Users

Platforms connecting buyers and sellers for goods, services, rentals, or freelance work.

## Key Capabilities

- Listing creation with image upload
- Search with filters, sorting, and category browsing
- Listing detail page with image gallery
- Seller profiles with reviews and ratings
- Buyer/seller messaging
- Saved/favorited listings
- Order and transaction status tracking

## Stack

Next.js + Convex + Clerk + Vercel + XState
