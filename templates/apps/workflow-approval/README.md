# Workflow Approval

Request-and-approval system template for structured multi-step review workflows.

## App Class

`workflow_approval_system`

## Target Users

Organizations that need structured request submission and multi-level approval flows (expense reports, leave requests, procurement, change requests).

## Key Capabilities

- XState-driven approval state machines
- Multi-step request submission wizard
- Review queue with role-based filtering
- Approve/reject flow with confirmation dialogs
- Status timeline per request
- Comment threads on requests
- Audit trail for every state transition
- Bulk approval for reviewers

## Stack

Next.js + Convex + Clerk + Vercel + XState
