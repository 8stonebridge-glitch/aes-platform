# Auth Layout

Layout template for authentication pages (sign in, sign up, forgot password).

## What It Provides

- Centered card layout
- Brand logo and tagline
- Split panel option (illustration + form)
- Clerk component integration slot
- Footer links (terms, privacy, help)
- Redirect logic after auth

## States Handled

- Sign in view
- Sign up view
- Forgot password view
- SSO redirect in progress
- Auth error (invalid credentials, expired link)
- Already authenticated (redirect)

## What Builder Fills

- Brand logo and tagline
- Illustration or background image
- Clerk appearance customization
- Post-auth redirect route
- Footer link URLs
- Custom OAuth provider configuration
