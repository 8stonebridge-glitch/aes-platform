# Marketing Layout

Layout template for public-facing marketing and landing pages.

## What It Provides

- Top navigation with transparent-to-solid scroll behavior
- Hero section slot
- Full-width content sections
- Footer with link columns and legal links
- CTA bar (sticky or inline)
- Mobile-responsive navigation

## States Handled

- Nav transparent (top of page)
- Nav solid (scrolled)
- Mobile nav open
- Mobile nav closed
- CTA bar visible (scroll trigger)

## What Builder Fills

- Navigation items and CTA button
- Hero content and background
- Footer link definitions
- Legal page links (privacy, terms)
- Brand assets (logo, colors)
- Social media links
