# Sidebar Layout

Layout template with a persistent left sidebar navigation.

## What It Provides

- Collapsible sidebar with icon and text nav items
- Grouped navigation sections with dividers
- Active route highlighting
- Mobile: slide-out drawer with overlay
- Header bar with user menu and org switcher
- Breadcrumb slot below header
- Main content area with scroll container

## States Handled

- Sidebar expanded (desktop default)
- Sidebar collapsed (icon-only mode)
- Sidebar open (mobile drawer)
- Sidebar closed (mobile default)

## What Builder Fills

- Navigation item definitions (label, icon, route, badge)
- Section grouping and ordering
- User menu items
- Org switcher integration details
- Footer content (version, help link)
