# Top Nav Layout

Layout template with a horizontal top navigation bar.

## What It Provides

- Fixed top navigation bar with logo and links
- Dropdown menus for grouped items
- Mobile: hamburger menu with slide-down panel
- User menu with avatar
- Main content area below nav
- Optional secondary nav (tabs below top bar)

## States Handled

- Nav default (desktop horizontal)
- Nav open (mobile menu expanded)
- Nav closed (mobile default)
- Dropdown open (submenu visible)
- Scrolled (shadow on nav bar)

## What Builder Fills

- Navigation item definitions (label, route, children)
- Logo and brand assets
- User menu items
- Secondary nav tab definitions (if used)
- CTA button in nav (if applicable)
