# Empty State

Page template for empty/zero-data states.

## What It Provides

- Centered illustration or icon
- Heading and description text
- Primary call-to-action button
- Optional secondary action
- Optional "getting started" checklist

## States Handled

- First-time user (onboarding prompt)
- No data yet (create first item)
- No results (adjust filters)
- Feature not enabled (upgrade prompt)

## What Builder Fills

- Illustration or icon selection
- Heading and description copy per context
- CTA button text and route
- Checklist items for onboarding variant
- Permission-aware CTA visibility
