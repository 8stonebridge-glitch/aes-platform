# Error Page

Page template for error states (404, 500, 403, etc.).

## What It Provides

- Error code and message display
- Contextual illustration or icon
- Action buttons (go home, go back, retry)
- Support contact link
- Error ID for support reference

## States Handled

- 404 Not Found
- 403 Forbidden (insufficient permissions)
- 500 Server Error
- Network Error (offline)
- Unknown Error (fallback)

## What Builder Fills

- Custom error messages per code
- Custom illustrations per error type
- Support contact URL or email
- Error reporting integration
- Suggested navigation links
